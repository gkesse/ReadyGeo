# ReadyGeo

**ReadyGeo** est une bibliothèque de fonctions C++ pour la résolution de problèmes de Géométrie. 
C'est une bibliothèque développée en C++ et maintenu par 
[Gerard KESSE](http://5.48.215.193:8855/presentation/ "Accédez à mon site web (ReadyDev)").

# Technologies

Le développement de la bibliothèque **ReadyGeo** a nécessité les technologies suivantes :

* Langage : **C++**
* IDE : **Visual Studio**
* Outil de Construction : **NMake, Cygwin**
* Editeur de Texte : **Notepad++**
* Concept Vecteur : **Longueur, Normal, Orthogonal**
* Figure Géométrique : **Droite, Cercle, Polygone**
* Concept Géométrie 2D/3D : **Equation, Intersection**
* Cible : **Windows, UNIX**
